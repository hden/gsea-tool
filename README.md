# GSEA helper
Extract leading-edge from GSEA reports
## Usage
In coffee-script:
```coffee-script
helper = require '../src/index'
data   = # GSEA data
result = helper.leadingEdge(data)
```
## Requirement
Underscore.js `npm install underscore`
